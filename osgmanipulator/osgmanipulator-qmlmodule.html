osgManipulator

The osgManipulator module extends the core scene graph to support 3D interactive manipulators

Importing

To use the types in this module, import the module with the following line:

import osgManipulator 2.0

Types

The following types are provided:

Dragger

Base type for draggers

RotateSphereDragger

Dragger for performing 3D rotation on a sphere
