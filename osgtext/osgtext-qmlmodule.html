osgText

The osgText module extends the core scene graph to support high quality text

Importing

To use the types in this module, import the module with the following line:

import osgText 2.0

Types

The following types are provided:

Font

Text style

FontImplementation

Base type for font implementation

Style

Text font

Text3D

3-dimensional text

TextBase

Base type for text Nodes
