osgQtQuick

Integration OpenSceneGraph library to Qt Quick.

Structure

This project provides one QML module for each OpenSceneGraph library.

OpenSceneGraph QML modules

osg

Contains QML types for osg library

osgDB

Contains QML types for osgDB library

osgGA

Contains QML types for osgGA library

osgManipulator

Contains QML types for osgManipulator library

osgText

Contains QML types for osgText library

osgViewer

Contains QML types for osgViewer library
